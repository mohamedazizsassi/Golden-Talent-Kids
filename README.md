# Golden Talents Kids 3.0

Landing page for the Golden Talents Kids 3.0 sponsorship campaign. Built with React, Vite, and Tailwind CSS.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide Icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

```text
src/
	components/    UI sections (Hero, Packs, Schedule, etc.)
	lib/           Utilities (WhatsApp link helper)
	index.css      Global styles and Tailwind layers
	main.jsx       App entry point
public/          Static assets (logos, images)
```

## Key Pages and Sections

- Hero and CTA
- Sponsorship packs
- Schedule and countdown
- Past editions and stats
- Why sponsor

## Asset Notes

- Place static logos and images in the public folder and reference them with absolute paths (for example, /logo-poly.png).

## Deployment

The site is a static Vite build. Any static host that serves the dist folder will work.
